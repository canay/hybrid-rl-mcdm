# Hybrid RL-MCDM v16

This repository contains the active v16 workspace for the Hybrid RL-MCDM
e-commerce recommendation study, accompanying the manuscript
*"Observable and Behavioral Signal Complementarity in E-Commerce
Recommendation: A Hybrid Entropy-Weighted TOPSIS and Tabular Q-Learning
Framework"* (under review at Expert Systems with Applications).

## Repository layout
- `manuscript/v16/`: active LaTeX source tree (`paper_v16.tex`),
  publication figures, and `cas-refs.bib`.
- `src/`: modular Python experiment code (active = v2 / v16).
- `data/`: synthetic product catalogs used by the bootstrap runs.
- `results/`: JSON artifacts generated by the active pipeline.
- `docs/`: methodology and writing notes.
- `tests/`: reproducibility smoke tests.
- `archive/`: superseded paper versions, legacy v15 source, and
  early exploratory experiments (Amazon Home & Kitchen pilot, etc.).
- `requirements.txt`: pinned Python package versions.

## Active experiment files
- `src/hybrid_rl_mcdm_v2.py`: full reproducible pipeline (data
  generation, 30-run primary bootstrap, lambda and GT-split ablation,
  concept drift, confidence-gated fusion, XAI signal decomposition).
- `src/v16_pacaon_verify.py`: empirical verification that the
  TOPSIS-only baseline is operationally equivalent to the
  Pacaon and Ballera (2024) entropy-EWM-TOPSIS recommender
  (Kendall tau = 1.000 across 30 seeds).
- `src/v16_learned_lambda.py`: oracle-bound static-versus-learned
  lambda analysis on the existing lambda ablation grid.
- `src/generate_figures_v16.py`: artifact-driven figure regeneration
  for the v16 manuscript.

## Results produced by the pipeline
- `results/v2_full_results.json` -- main experiment artifacts.
- `results/v2_supplementary.json` -- profile-level NDCG and reward-shaping.
- `results/v16_pacaon_verify.json` -- Pacaon-Ballera operational equivalence.
- `results/v16_learned_lambda.json` -- learned-versus-static lambda summary.
